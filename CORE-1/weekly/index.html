week 1

Since the goal of this assignment was material exploration, I made sure to experiment with a variety of markmaking tools and mediums. I used ballpoint pen, fineliner, brushpens, and a cutter.

I image traced a previous artwork (based off of turtle shells) to get a vector pattern in Illustrator. I tried using a regular water-based pen and a paint pen on the duralar, and the paint pen was significantly more successful as the fineline smudged under the plotter. I would be interested in using the plotter technique to create negatives that I could use with cyanotype, as I have found that paint pens are fairly successful in blocking the light neeeded to make prints.

For the fish, I used a combination of Photoshop and Illustrator to make two different files for the red and blue filtered versions. To create the dual toned effect, I simply left the paper in the machine and changed out the pen. One disappointing thing about using the brush pens was that I wasn't able to use the full range of line weight that you can usually achieve, mainly because the force of the plotter would inevitably push the pen to the same depth each time.

week 2

Instead of using a pre-existing program to generate parameterized designs, I decided to experiment with coding my own procedural 'editor'. I decided to use p5js due to familiarity since I didn't want to learn a new (probably more suitable) program or language, when p5js is meant for image generation.

The main challenge is that p5js mainly uses raster, so I had to search through other work to find plugins and code that would work to not only generate vector images but also save them as an SVG. I also had the added challenge of having to generate the vectors and anchor points through code instead of intuitively placing them like on Illustrator. I ended up having to generate the individual vectors and points which could then be input into the existing bezier function. As far as I know, p5js only allows for cubic bezier curves, so I had to turn them into quadratic beziers (for the sake of simplicity) by equating the anchor points. I then had the added challenge of having to repeat these calculations for each petal at a different angle, all of which would change depending on the variable parameters I set.

The final result was very similar to what I had in mind! The flower pattern can be modified in size, line thickness, and petal number, and can then be exported as an SVG file which can be opened in Illustrator and edited like any other vector file. The plotter designs are honestly secondary -- I added a couple of grid options but didn't really have the time or energy to implement them as thoroughly as I would have liked.

week 4

This week we just had to select photos which would later be modified or used to create a book. I chose images of my dog through stages of his life, as well as used images from dog toys and baby toy companies, thinking about the parallels of puppyhood and childhood. I was also curious about the assignment's approach to paramtetrizing image collection, and how we parametrize and ritualise daily life throughout stages of growth.

week 5

The assignment this week was to create 16 or more pages using images that are chosen in a procedural way. I chose to generate the images and order using p5js, as I was interested in the way images could be processed differently on each iteration/time I ran the code.

I made this code with the assistance of ChatGPT, although it is not my preferred method, and was honestly surprised at how effective it is at generating simple code. Each time i run the program, it will alter the images a random amount using a variety of preset filters on p5js, and then will sort the images by the percentage of a chosen color.

I knew that these experimental assignments were working up to a physical book, which to me is always a tactile experience, so when thinking about images I wanted to push code and web as the medium. I think that the images came out in a very interesting and satisfyingly random way which really embodies new media aesthetics, but also has a sense of degradation due to the filtering.
